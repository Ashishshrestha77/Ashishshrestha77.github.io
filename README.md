ASHISH SHRESTHA – PORTFOLIO WEBSITE

---

OVERVIEW
This is a personal portfolio website showcasing my work as a content creator, videographer, and photographer. The site highlights my video projects, client photography, personal work, and professional background.

---

FEATURES

* Responsive design (desktop, tablet, and mobile)
* Video project showcase with YouTube integration
* Photography portfolio with client and personal work sections
* Interactive image gallery with fullscreen modal view
* Smooth scrolling navigation
* Animated scroll reveal effects
* Premium UI with hover interactions
* Mobile-friendly navigation (hamburger menu)

---

TECHNOLOGIES USED

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Font Awesome (for icons)

---

HOW TO RUN

1. Download or clone the repository
2. Open the project folder
3. Double-click "index.html"
   OR
   Right-click → Open with browser

No additional setup required.

---

CUSTOMIZATION

To update content:

* Edit text inside index.html
* Replace images in the project folder
* Update video links (YouTube URLs)

To modify design:

* Edit styles inside style1.css
